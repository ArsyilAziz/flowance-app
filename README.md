# 🌿 Flowance

A clean and modern personal finance web application built with Laravel.

Flowance helps you track your income and expenses with clarity, calmness, and control — without overwhelming dashboards.

---

## ✨ Preview

> A minimalist editorial finance experience designed for clarity and emotional comfort.

- Dashboard overview (balance, income, expense)
- Smooth transaction tracking
- Clean category system
- Editorial UI inspired by modern fintech

---

## 🎯 Features

- 💰 Track income & expenses
- 🗂️ Category management
- 📜 Transaction history
- 📊 Financial dashboard
- 🎨 Clean UI (Tailwind CSS)

---

## 🧠 Philosophy

Flowance is not just a finance tracker.

It is designed with:
- **Clarity over complexity**
- **Calmness over clutter**
- **Control over chaos**

Inspired by editorial design and mindful finance principles.

---

## 🛠 Tech Stack

- Laravel (Backend)
- Tailwind CSS (Frontend)
- MySQL (Database)
