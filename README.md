# Flowance

A clean and modern personal finance web application built with Laravel.

Designed with a focus on clarity, simplicity, and mindful financial tracking.

## ✨ Features
- Income & expense tracking
- Category management
- Transaction history
- Financial dashboard

## 🛠 Tech Stack
- Laravel
- Tailwind CSS
- MySQL

## 🎨 Design Philosophy
Minimal, editorial, and calm — helping users stay in control of their finances.
